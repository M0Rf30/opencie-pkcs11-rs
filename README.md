# opencie-pkcs11

Rust bindings for [libopencie-pkcs11](https://github.com/M0Rf30/opencie-pkcs11) — a PKCS#11 interface for the Italian CIE (Carta d'Identità Elettronica / Electronic Identity Card).

This crate provides both low-level FFI bindings and safe Rust wrappers for:
- **PKCS#11 standard interface**: Token operations, sessions, signing, encryption, etc.
- **CIE-specific extensions**: Card enrollment, PIN management, PDF signing, signature verification
- **Sign SDK**: Advanced signing and verification operations

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
opencie-pkcs11 = "0.1"
```

## Requirements

### Runtime
- **libopencie-pkcs11** shared library must be installed on your system
- Linux: Install from [releases](https://github.com/M0Rf30/opencie-pkcs11/releases) or build from source
- macOS/Windows: See [upstream documentation](https://github.com/M0Rf30/opencie-pkcs11#building)

### Build
- **Clang/LLVM** (required by bindgen)
  - Linux: `apt-get install libclang-dev` or `yum install clang-devel`
  - macOS: `xcode-select --install`
  - Windows: Install LLVM from [llvm.org](https://releases.llvm.org/)

## Usage

### PKCS#11 Standard Operations

```rust
use opencie_pkcs11::pkcs11;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Initialize PKCS#11 library
    pkcs11::initialize()?;
    
    // Get available slots
    let slots = pkcs11::get_slot_list(true)?;
    println!("Found {} slot(s)", slots.len());
    
    // Open a session
    let session = pkcs11::open_session(
        slots[0],
        opencie_pkcs11::ffi::CKF_SERIAL_SESSION | opencie_pkcs11::ffi::CKF_RW_SESSION
    )?;
    
    // Login with PIN
    pkcs11::login(session, opencie_pkcs11::ffi::CKU_USER, "12345678")?;
    
    // ... perform cryptographic operations ...
    
    // Logout and cleanup
    pkcs11::logout(session)?;
    pkcs11::close_session(session)?;
    pkcs11::finalize()?;
    
    Ok(())
}
```

### CIE Extension Functions

```rust
use opencie_pkcs11::cie;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let pan = "1234567890123";
    let pin = "12345678";
    
    // Check if card is enrolled
    if !cie::is_enabled(pan) {
        // Enroll the card
        cie::enable(pan, pin)?;
    }
    
    // Sign a PDF document
    cie::sign(
        "input.pdf",    // input file
        "PDF",          // signature type
        pin,            // PIN
        pan,            // PAN
        0,              // page (0-based)
        100.0, 100.0,   // x, y position
        200.0, 50.0,    // width, height
        None,           // optional image path
        "signed.pdf"    // output file
    )?;
    
    // Verify a signed document
    let signature_count = cie::verify(
        "signed.pdf",
        None,  // proxy address
        0,     // proxy port
        None   // proxy credentials
    )?;
    println!("Found {} signature(s)", signature_count);
    
    // Extract original file from .p7m envelope
    cie::extract_p7m("document.pdf.p7m", "document.pdf")?;
    
    Ok(())
}
```

### Sign SDK Operations

```rust
use opencie_pkcs11::sign;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Initialize signing context
    let ctx = sign::sign_init();
    
    // Configure signing options
    sign::sign_set_string(&ctx, 7, "input.pdf")?;   // CIE_SIGN_OPT_INPUTFILE
    sign::sign_set_string(&ctx, 8, "signed.pdf")?;  // CIE_SIGN_OPT_OUTPUTFILE
    sign::sign_set_string(&ctx, 3, "12345678")?;    // CIE_SIGN_OPT_PIN
    
    // Execute signing
    sign::sign_sign(&ctx)?;
    
    // Cleanup
    sign::sign_cleanup(ctx)?;
    
    Ok(())
}
```

## Modules

- **`ffi`**: Raw PKCS#11 FFI bindings (generated by bindgen)
- **`cie_ffi`**: Raw CIE extension FFI bindings
- **`sign_ffi`**: Raw Sign SDK FFI bindings (manual bindings)
- **`pkcs11`**: Safe Rust wrappers for PKCS#11 operations
- **`cie`**: Safe Rust wrappers for CIE-specific functions
- **`sign`**: Safe Rust wrappers for Sign SDK

## Error Handling

All operations return `Result<T, pkcs11::Error>` where `Error` wraps the underlying PKCS#11 `CK_RV` return code:

```rust
use opencie_pkcs11::pkcs11::{Error, Result};

fn example() -> Result<()> {
    match pkcs11::initialize() {
        Ok(()) => println!("Initialized"),
        Err(Error(code)) => eprintln!("Error: 0x{:08x}", code),
    }
    Ok(())
}
```

## Callback Support

**Note**: Callback support for CIE operations (progress and completion callbacks) is not yet implemented. The safe wrappers pass `NULL` for callback parameters. Full callback support will be added in a future release.
